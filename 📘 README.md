# WG‑TURN Server Installer  
Полностью автоматизированный установщик **WireGuard + VK TURN Proxy**  
с поддержкой **TUI‑меню**, генерацией клиентов, QR‑кодов и полной очисткой.

---

## 🚀 Возможности

### 🟢 Установка WireGuard
- автоматическая генерация ключей  
- создание `wg0.conf`  
- настройка NAT и iptables  
- автоматический выбор сетевого интерфейса  
- проверка занятости порта  
- поддержка Debian 11/12 (с авто‑фиксом репозиториев)

### 🟢 Установка VK TURN Proxy
- автоматическое определение архитектуры (x86_64 / ARM64)  
- скачивание нужного бинарника  
- создание systemd‑юнита  
- автозапуск и автоперезапуск  

### 🟢 Управление клиентами
- добавление клиентов  
- удаление клиентов  
- просмотр списка  
- генерация QR‑кодов  
- хранение конфигов в `~/wg-clients/`

### 🟢 TUI‑меню (`wg-menu`)
Интерактивное меню управления:

1. Добавить клиента

2. Удалить клиента

3. Показать список клиентов

4. Показать QR клиента

5. Перезапустить WireGuard

6. Статус сервисов

7. Удалить VK TURN Proxy

8. Выход


### 🟢 Полная очистка (`wg-clean`)
Удаляет:

- WireGuard  
- VK TURN Proxy  
- все клиенты  
- все конфиги  
- все скрипты  
- TUI‑меню  
- systemd‑юниты  
- пакеты wireguard  
- сам себя

## 📥 Установка

### ✔ Рекомендуемый интерактивный запуск (порт спрашивается)
```bash
bash <(curl -s https://raw.githubusercontent.com/Vista-21/WG-TURN-server-installer/main/install-server-wg-turn.sh)
```

✔ Альтернативный способ
```bash
curl -O https://raw.githubusercontent.com/Vista-21/WG-TURN-server-installer/main/install-server-wg-turn.sh
chmod +x install-server-wg-turn.sh
./install-server-wg-turn.sh
```

✔ Запуск через pipe (порт НЕ спрашивается)
```bash
curl -s https://raw.githubusercontent.com/Vista-21/WG-TURN-server-installer/main/install-server-wg-turn.sh | bash
```

# 📂 Где хранятся клиенты

Все конфиги создаются в:

`~/wg-clients/
`

Пример:
`~/wg-clients/main_test.conf
`
`~/wg-clients/user_test#1.conf
`
`~/wg-clients/user_test#2.conf
`

# 🧰 Команды управления

| Команда | Описание |
| --- | --- |
| ``wg-add-client``<name></name>`` | Создать клиента |
| ``wg-del-client``<name></name>`` | Удалить клиента |
| ``wg-peers`` | Показать список клиентов |
| ``wg-menu`` | Открыть TUI‑меню |
| ``vk-turn-clean`` | Удалить VK TURN Proxy |
| ``wg-clean`` | Полная очистка сервера |
